# RGA_APP_3.0
Sistema web criado para atender uma necessidade real do trabalho da minha esposa. Permite registrar horas trabalhadas, calcular ganhos por produção e gerar relatórios. Desenvolvido em JavaScript, HTML e CSS, com manipulação do DOM, dashboard dinâmico, backups em localStorage/JSON e exportação em PDF e Word. Projeto em evolução (v3.0).
